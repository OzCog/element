Element CSS

Saner defaults for HTML elements.

Element is a CSS design / framework that focuses on styling basic HTML elements. In contrast to most design frameworks which utilize a lot of nested <div> elements and inline classes, Element does away with that, making it quicker and easier to create decent-looking static web sites from scratch that are easy to edit and manage with just a text editor.

Think of it as a Markdown for HTML. Instead of writing markdown and then converting it into nasty HTML, we eliminate the Markdown part and just create sites directly in clean, readable HTML. This also lets us do things Markdown can't really do well, like grids.

This project is still in it's early stages. If you have any ideas or improvements, please feel free to chime in.

Based on ungrid, Element CSS wires up <grid-row> to be a row and <grid-col> to be a column, using a clever CSS hack to make an easy, auto-sized, responsive grid that Just Works.

Code

<code> elements used in a <p> will be inline-block: echo('hello world!'). Code tags outside of a <p> will be blocks, with code indenting automatically fixed (no <pre> indenting gymnastics needed).

<code> const user = { name: 'Commander Riker', username: 'hotjazzhotternights1337' } console.log(user.name) </code>

If you need to embed HTML code examples, you can use the <xmp> tag, which makes it so you don't need to encode escapes. Sadly, the HTML5 spec is trying to obsolete the <xmp> tag even though it's the only way to make this work, but all browsers seem to be supporting it anyways. Use at your own risk.

<!doctype html> <html> <head> <title>Hello!</title> </head> <body> <h1>I didn't have to escape any of this (inspect the source code).</h1> </body> </html>

If you want syntax highlighting, just embed highlight.js and the highlighting style you want and it's automatically configured.

How to Use

Use this template to get started:

<!doctype html> <html class="element"> <head> <link rel="stylesheet" href="https://elementcss.neocities.org/dist/element-0.0.1.min.css"> <script src="https://elementcss.neocities.org/dist/element-0.0.1.min.js"></script> <title>Hello!</title> </head> <body> <header> <nav> <a href="one.html">Link One</a> <a href="two.html">Link Two</a> </nav> <h1>Site Name</h1> <strong>Subtitle for site</strong> </header> <p>Okay, you're all set!</p> </body> </html>
