Element CSS

An experimental CSS "framework" for static HTML coders.

Element CSS is the CSS framework for coders that want to go back to just writing simple, static HTML that looks presentable. In contrast to frameworks like Bootstrap that require a lot of nested <div> elements with inline classes, Element CSS focuses on just using basic HTML elements to make a web site. The result is the ability to quickly and easily draft up document-oriented static web sites that are easy to edit and manage with just a text editor. View the source code of this site, and you'll see what I mean.

Think of it as a Markdown for HTML. Instead of writing markdown and then converting it into unreadable HTML, we eliminate the Markdown part and just create sites directly in readable HTML. This also lets us do things Markdown can't really do well, like grids.

Code

<code> elements used in a <p> will be inline-block: echo('hello world!'). Code tags outside of a <p> will be blocks, with code indenting automatically fixed (no <pre> indenting gymnastics needed).

const user = { name: 'Commander Riker', age: 45, username: 'hotluvin33' } console.log(user.name)

If you need to embed HTML code examples, use the <xmp> tag, which makes it so you don't need to encode escapes. Sadly, the HTML5 spec is trying to obsolete the <xmp> tag even though it's the only way to make this work, but all browsers seem to be supporting it anyways. Use at your own risk.

<!doctype html> <html> <head> <title>Hello!</title> </head> <body> <h1>I didn't have to escape any of this (inspect the source code).</h1> </body> </html>

If you want syntax highlighting, just embed highlight.js and the highlighting style you want and it's automatically configured.

How to Use

Use this template to get started:

<!doctype html> <html> <head> <link rel="stylesheet" href="element.css"> <script src="element.js"></script> <title>Hello!</title> </head> <body> <header> <nav> <a href="one.html">Link One</a> <a href="two.html">Link Two</a> </nav> <h1>Site Name</h1> <strong>Subtitle for site</strong> </header> <p>Okay, you're all set!</p> </body> </html>
